## 下载

### Docker

```
docker pull apachecn0/flygon-python-zh
docker run -tid -p <port>:80 apachecn0/flygon-python-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-python-zh
flygon-python-zh <port>
# 访问 http://localhost:{port} 查看文档
```
