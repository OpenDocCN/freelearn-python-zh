# 前言

Python 最初是由 Guido Von Rossum 开发的一个有趣项目，并以他最喜欢的节目《蒙提·派森的飞行马戏团》命名。它于 1991 年开发，但直到 2008 年左右才开始流行。Python 的流行得益于 Google，它开发了多个使用 Python 的平台。最近，Python 因云计算、DevOps、数据科学、数据分析、机器学习和自然语言处理而受到普及。随着越来越多的数据采集和处理，人们希望进入新的工作角色，这些角色需要基本的编程技能，而 Python 完美地适合所有工作类别。《7 天学会 Python》就是为了给这样的人提供一个简单的方法，在 7 天内学习和掌握 Python 的基础知识。本书涵盖了理解 Python 语言工作原理所需的基本和必要概念。本书适合所有类型的读者和学习者。它也适用于有经验的人士进行复习。我们相信，我们已经尽可能多地涵盖了内容，使其成为一本可以在 7 天内完成的书籍；然而，我们认为仅仅阅读本书是不够掌握编程技能的。要达到精通，还需要更多。我们希望您喜欢阅读本书，并将其作为一本优秀的教材。

# 本书涵盖内容

第一章，*Python 入门*，为您提供了 Python 编程语言的基础背景，包括安装说明和基本命令。

第二章，*类型变量和运算符*，为您提供了关于 Python 中变量类型和使用的各种运算符的想法。

第三章，*字符串*，探讨了 Python 中各种字符串和字符串操作。

第四章，*列表*，介绍了 Python 中的一种重要集合类型，即列表，以及可以在列表上执行的所有相关操作。

第五章，*字典*，涵盖了 Python 中另一种重要的集合类型，即字典，并处理了字典的基本知识和对字典的各种操作。

第六章，*控制语句和循环*，讨论了各种控制语句，如 if 语句和循环（for、while 等），这些可以在 Python 中编写代码。

第七章，*函数和变量的作用域*，为您提供了关于用户定义或内置的函数的想法。本章讨论了函数的基本知识和各种操作。此外，本章还为您提供了关于代码块特定作用域内变量有效性的想法。

第八章，*模块和包*，讨论了 Python 中可用的各种模块和包，以及如何创建自己的包和模块。

第九章，*文件处理和异常*，分为两部分；第一部分处理不同的文件处理操作，第二部分处理异常处理。

第十章，*集合*，为你提供了 Python 中不同类型和子类型集合的入门知识，并解释了可以对它们执行的基本操作。

第十一章，*类和对象*，最终为你介绍了一些 Python 中可用的面向对象编程概念。

# 你需要为此书准备什么

对于这本书，你需要在你的机器上安装 Python 2.7x 版本，以及一个简单的文本编辑器（记事本或 Notepad++）。所有示例都旨在在 Python 2.7 版本上运行，并且不会在 Python 3.x 版本上工作。
