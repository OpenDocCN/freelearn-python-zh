Preface

Qt is one of the most widely used and flexible frameworks for GUI application development. This book will introduce you to the development of Python-based Qt GUI applications with Python bindings such as PyQt and PySide. We will start by describing PyQt and PySide GUI elements, and this will be used as a reference for the rest of the book. On completion of the book, you will be able to create beautiful Python GUI applications from scratch.

Who this book is for

This book is for Python developers who want to develop GUIs and cross-platform applications that are modern, responsive, and attractive. No prior knowledge of Qt or QML is required.

What this book covers

Chapter 1, Introduction to Python and Qt, describes the basics of Qt and compares it with Python. It also discusses the fields of implementing various GUIs with Python.

Chapter 2, An Overview of QML, introduces the Qt modeling language.

Chapter 3, Qt Quick Library, discusses the implementation of the Qt Quick library.

Chapter 4, Getting Started with PyQt and PySide, introduces PyQt, PySide and some of the most useful methods for GUI application construction, along with the concept of styling.

Chapter 5, Using QWidget and Main Window, shows how to use widgets and frames in the GUI application and discusses the features of the widgets and frames in PyQt5, PySide2 and their functionalities.

Chapter 6, Using Frames, Labels, and Text Fields, deals with PyQt5 and PySide2 labels, text fields, combo boxes, and related element features.

Chapter 7, Using Combo Boxes and List Views, discusses combo boxes, list views, and tables in PyQt5 and PySide2, along with the functionality of these elements and features.

Chapter 8, Implementing Buttons, describes the implementation of buttons and other control elements in the PyQt5 and PySide2 GUI application.

Chapter 9, The Basics of Graphics, describes PyQt and PySide classes for working with graphics, drawing and painting, graphic views, graphic scenes, and other visualization components.

Chapter 10, Graphics Representation, concentrates on PyQt and PySide classes that represent the graphics used in the application and other visualization components.  

Chapter 11, Graphical Effects and Multimedia, focuses on PyQt and PySide classes that represent the graphics in the application, graphic animations, and using the additional graphics tools in the application. 

Chapter 12, Files, Printers, and Other Dialogs, deals with PyQt and PySide classes for working with files, such as open, save, and create. It also focuses on input dialogs and message boxes.

Chapter 13, Creating Layouts, concentrates on PyQt and PySide classes for positioning elements and explains how to create layouts to make your GUI application responsive and resolution independent.

Chapter 14, Implementing Databases, shows how to create databases and work with SQL/NoSQL, and adding persistence.

Chapter 15, Signals, Slots, and Event Handlers, introduces signals and slots and explains how to add PyQt and PySide elements along with the functional features and possibilities for communication between elements. Also, it describes the events in PyQt and PySide GUI and how to implement event handlers.

Chapter 16, Threading and Multiprocessing, shows the implementation of the threading and multiprocessing approaches with PyQt and PySide GUI creation. Also, it focuses on the main principles of building large-scale applications, along with and database handling.

Chapter 17, Completing the Application, Embedded Devices, and Tests, completes the application, carries out testing, and describes the basics of deploying the application to embedded platforms.
