Using QWidget and Main Window

From this chapter onward, the graphical elements of a GUI application will be explored. The methods relating to these visual objects and other functionality will be considered. When a developer starts to build the GUI, they need to fill this application with different graphical objects, including frames, labels, fields, buttons, and many more control elements. In this chapter, we will be covering the basic fields for graphical components or the places when they can be visualized. In Qt/C++, the QtWidgets class commonly provides this functionality and is available with the PyQt5 and PySide2 Python bindings.

 This chapter will cover the following topics:

  • QWidget
  • QMainWindow

QWidget

This class is the base class for all the user interface objects provided by the QtWidgets module. As was explained in the PySide2 documentation (https://doc.qt.io/qtforpython/PySide2/QtWidgets/QWidget.html), this is the atom of the user interface. This class has the following inheritance scheme:

It also has a parent for almost all graphical elements, additional widgets, and so on. The declaration syntax is as follows:

widget = QtWidgets.QWidget()

Functions of QWidget

QWidget inherits QObject and QPaintDevice and improves functionality by means of the following functions.

set

These are functions that are related to the setting of the parameters/properties in relation to the widget, including the setting of style, width, or functionality:

widget.setAcceptDrops(bool): If this is set to True, the widget will accept the drop event.

widget.setAccessibleDescription("description"): This provides an accessible description for this widget.

widget.setAccessibleName("Name"): This provides an accessible name for this widget.

widget.setAttribute(QtCore.Qt.WidgetAttribute, bool): This sets the attribute on this widget if the second parameter is True; otherwise, it clears the attribute.

widget.setAutoFillBackground(bool): If the parameter is True, the widget's background will be filled automatically.

widget.setBackgroundRole(QtGui.QPalette.ColorRole): This sets the background role of the widget to the parameter role of the QtGui.QPalette.ColorRole type.

widget.setBaseSize(QtCore.QSize): This sets the widget's base size to the parameter of the QtCore.QSize type.

widget.setBaseSize(basew, baseh): This sets the widget's base size to parameters where basew (base width) and baseh (base height) are integer values.

widget.setContentsMargins(QtCore.QMargins): This sets the margins of the QtCore.QMargins type around the widget's contents.

widget.setContentsMargins(left, top, right, bottom): This sets the margins as integer values for each side around the widget's contents.

widget.setContextMenuPolicy(QtCore.Qt.ContextMenuPolicy): This sets the context menu policy for this widget, which means how this widget displays a context menu.

widget.setCursor(QtGui.QCursor): This sets the current cursor of the QtGui.QCursor type—the widget's cursor.

widget.setDisabled(bool): If the parameter is True, this disables the widget input events; otherwise, the input events are enabled.

widget.setEnabled(bool): If the parameter is True, this enables widget input events that were disabled.

widget.setFixedHeight(h): This sets a fixed height with the integer value h (height of the widget); the height of the widget will not change in the layout system.

widget.setFixedWidth(w): This sets a fixed width with the integer value w (width of the widget); the width of the widget will not change in the layout system.

widget.setFixedSize(QtCore.QSize): This sets the widget's fixed size of the QtCore.QSize type; the size of the widget will not change in the layout system.

widget.setFixedSize(w, h): This sets the widget's fixed size with integer values as w (width) and h (height); the size of the widget will not change in the layout system.

widget.setFocus(): This sets the keyboard input focus to this widget if this widget, or one of its parents, is the active window.

widget.setFocus(QtCore.Qt.FocusReason): This sets the keyboard input focus to this widget if this widget, or one of its parents, is the active window. The reason for the QtCore.Qt.FocusReason type being used is to provide an explanation regarding what caused the widget to achieve focus.

widget.setFocusPolicy(QtCore.Qt.FocusPolicy): This sets the widget's focus policy; that is, the way the widget accepts keyboard focus.

widget.setFocusProxy(QtWidgets.QWidget): This sets the widget's focus proxy to the parameter of the QtWidgets.QWidget type.

widget.setFont(QtGui.QFont): This sets the widget's font.

widget.setForegroundRole(QtGui.QPalette.ColorRole): This sets the foreground role of the widget to the role in the parameter.

widget.setGeometry(QtCore.QRect): This sets the widget's geometry relative to its parent, excluding the window frame.

widget.setGeometry(x, y, w, h): This sets the widget's geometry relative to its parent, excluding the window frame, with the parameters x (x axis), y (y axis), w (width), and h (height).

widget.setGraphicsEffect(QtWidgets.QGraphicsEffect): This sets the widget's graphics effect.

widget.setHidden(bool): If this parameter is True, it sets the widget as hidden.

widget.setInputMethodHints(QtCore.Qt.InputMethodHints): This sets the input method hints that the widget will have.

widget.setLayout(QtWidgets.QLayout): This sets the layout manager for this widget. The widget can only have one layout manager. If setting a new one, the previous version needs to be deleted.

widget.setLayoutDirection(QtCore.Qt.LayoutDirection): This sets the layout direction for this widget.

widget.setLocale(QtCore.QLocale): This sets the locale for this widget.

widget.setMask(QtGui.QBitmap): This only sets the pixels of the widget, for which a bitmap parameter has a corresponding 1 bit, to be visible.

widget.setMask(QtGui.QRegion): This only sets the parts of the widget that overlap the region parameter to be visible.

widget.setMaximumHeight(maxh): This sets the maximum height of the widget with the integer value, maxh (the maximum height).

widget.setMaximumWidth(maxw): This sets the maximum width of the widget with the integer value, maxw (the maximum width).

widget.setMaximumSize(QtCore.QSize): This sets the widget's maximum size to the parameter of the QtCore.QSize type.

widget.setMaximumSize(maxw, maxh): This sets the widget's maximum size, with integer values as maxw, the maximum width, and maxh—the maximum height—parameters.

widget.setMinimumHeight(minh): This sets the minimum height of the widget with an integer value of minh, the minimum height.

widget.setMinimumWidth(minw): This sets the minimum width of the widget with an integer value of minw, the minimum width.

widget.setMinimumSize(QtCore.QSize): This sets the widget's minimum size to the parameter of the QtCore.QSize type.

widget.setMinimumSize(minw, minh): This sets the widget's minimum size with integer values as minw, the minimum width, and minh—the minimum height—parameters.

widget.setMouseTracking(bool): If this parameter is True, it sets the mouse tracking that's available for this widget.

widget.setPalette(QtGui.QPalette): This sets the widget's palette that is used by the widget's style when rendering standard components.

widget.setParent(QtWidgets.QWidget): This sets the parent from the parameter of the QtWidgets.QWidget type for this widget.

widget.setParent(QtWidgets.QWidget, QtCore.Qt.WindowFlags): This sets the parent from the first parameter of the QtWidgets.QWidget type for this widget, with flags as the second parameter.

widget.setShortcutAutoRepeat(id, bool): If the second parameter is True, the auto-repeat of the shortcut with the given id is enabled; otherwise, it is disabled.

widget.setShortcutEnabled(id, bool): If the second parameter is True, the shortcut with the given id is enabled; otherwise, it is disabled.

widget.setSizeIncrement(QtCore.QSize): This sets the size increment of the QtCore.QSize type for this widget, or the steps by means of which the size will change when the user resizes the window.

widget.setSizeIncrement(w, h): This sets the size increment with w, the width, and h, the height, parameters for this widget, or the steps by means of which the size will change when the user resizes the window.

widget.setSizePolicy(QtWidgets.QSizePolicy): This sets the size policy of the QtWidgets.QSizePolicy type for this widget.

widget.setSizePolicy(horizontal, vertical): This sets the size policy with horizontal, of the QtWidgets.QSizePolicy.Policy type, and vertical, of the QtWidgets.QSizePolicy.Policy type, for this widget.

widget.setStatusTip("Status tip"): This sets the widget's status tip.

widget.setStyle(QtWidgets.QStyle): This sets the widget's GUI style of the QtWidgets.QStyle type as styling.

widget.setStyleSheet("{property: value;}"): This sets the widget's style sheet, which contains a textual description of customizations to the widget's style.

widget.setTabletTracking(bool): If this parameter is True, tablet tracking will be enabled, or the widget will receive tablet move events even while hovering in proximity; otherwise, it is False.

widget.setTabOrder(QtWidgets.QWidget, QtWidgets.QWidget): This incorporates the widget of the second parameter after the widget of the first parameter in the focus order.

widget.setToolTip("Tool tip"): This sets the tooltip for this widget.

widget.setToolTipDuration(msec): This sets the duration for the tooltip—the time in msec (milliseconds) as an integer value when the tooltip will be visible.

widget.setUpdatesEnabled(bool): If this parameter is True, the widget's updates are enabled, the widget receives paint events, and has a system background. By default, updates are enabled.

widget.setVisible(bool): If this parameter is True, the widget will be visible if it was previously invisible; otherwise, the widget will be invisible.

widget.setWhatsThis("What's this"): This sets the widget's what's this help text.

widget.setWindowFilePath("File/path/to/the/widget"): This sets the file path associated with this widget.

widget.setWindowFlag(QtCore.Qt.WindowType, bool): This sets the window flag of the QtCore.Qt.WindowType type on this widget if the second parameter is True; otherwise, it clears the flag.

widget.setWindowFlags(QtCore.Qt.WindowFlags): This sets the window flags on this widget.

widget.setWindowIcon(QtGui.QIcon): This sets the icon for the window of this widget.

widget.setWindowIconText("Icon text"): This sets the icon's text for the window of this widget.

widget.setWindowModality(QtCore.Qt.WindowModality): This sets the behavior of a modal window, or which windows are blocked by the modal widget.

widget.setWindowModified(bool): If this parameter is True, it sets the window as modified.

widget.setWindowOpacity(float): This sets opacity in relation to the widget's window, where 0.0 is a completely transparent window, and 1.0 is a completely opaque window.

widget.setWindowRole(unicode): This sets Unicode string as the window's role to role; this only makes sense for Windows running on X11.

widget.setWindowState(QtCore.Qt.WindowStates): This sets the window state, such as QtCore.Qt.WindowMinimized, QtCore.Qt.WindowMaximized, QtCore.Qt.WindowFullScreen, or QtCore.Qt.WindowActive.

widget.setWindowTitle("Window Title"): This sets the title for the widget's window.

is/has

These are functions that return a Boolean value (bool) related to the widget states, for example, to check the current state:

widget.isActiveWindow(): This returns True if the widget's window is the active window.

widget.isAncestorOf(QtWidgets.QWidget): This returns True if the widget is a parent for the child parameter.

widget.isEnabled(): This returns True if the widget is enabled.

widget.isEnabledTo(QtWidgets.QWidget): This returns True if this widget would become enabled, if the parameter of the parent widget within the same window were to be enabled.

widget.isFullScreen(): This returns True if the widget is shown in fullscreen mode; otherwise, it returns False.

widget.isHidden(): This returns True if the widget is hidden; otherwise, it returns` False.

widget.isLeftToRight(): This returns True if the widget has a left-to-right direction.

widget.isMaximized(): This returns True if the widget's window is maximized.

widget.isMinimized(): This returns True if the widget's window is minimized.

widget.isModal(): This returns True if the widget's window is a modal widget.

widget.isRightToLeft(): This returns True if the widget has a right-to-left direction.

widget.isTopLevel(): This returns True if the widget is a top-level window.

widget.isVisible(): This returns True if the widget is visible.

widget.isVisibleTo(QtWidgets.QWidget): This returns True if the widget would become visible, if the parameter of the parent widget within the same window were to be shown.

widget.isWindow(): This returns True if the widget is the independent window; otherwise, it returns False.

widget.isWindowModified(): This returns True if the window content has unsaved changes.

widget.hasFocus(): This returns True or False depending on whether the widget keyboard has input focus or not.

widget.hasHeightForWidth(): If the widget's preferred height depends on its width, it will return True; otherwise, it returns False.

widget.hasMouseTracking(): This returns True if mouse tracking is enabled for the widget; otherwise, it returns False.

widget.hasTabletTracking(): This returns True if tablet tracking is enabled or the widget receives tablet move events, even while hovering in the proximity; otherwise, it returns False.

window

These are functions that make changes to the current widget visualization, such as closing and hiding:

widget.close(): This closes the widget and returns True.

widget.hide(): This hides the widget.

widget.show(): This shows the widget and its child.

widget.showFullScreen(): This shows the widget in full-screen mode.

widget.showMaximized(): This shows the widget in maximized mode.

widget.showMinimized(): This shows the widget in minimized mode.

widget.showNormal(): This shows the widget in normal mode.

widget.move(QtCore.QPoint): This moves the widget to the position of the QtCore.QPoint type.

widget.move(x, y): This moves the widget to the position of the x (X axis) or y (Y axis) parameters.

widget.resize(QtCore.QSize): This resizes the widget with the parameter of the QtCore.QSize type.

widget.resize(w, h): This resizes the widget with w (width) and h (height) parameters as integer values.

widget.scroll(dx, dy): This scrolls the widget, including its children, by dx pixels to the right, and dy pixels downward. dx and dy may be negative.

widget.scroll(dx, dy, QtCore.QRect): This scrolls the rectangle of the widget, excluding its children, by dx pixels to the right, and dy pixels downward. dx and dy may be negative.

QMainWindow

This class provides the main window that can be implemented with GUI applications. The QMainWindow has either a single document interface (SDI) or multiple document interface (MDI) that can be created by adding QMdiArea as the central widget. The most popular examples of MDI applications are the GNU Image Manipulation Program (GIMP) and Photoshop, which represent similarly displayed documents (Windows) instead of one (SDI). In both the SDI and MDI types of application, the central widget needs to be added. The declaration syntax for the main window in the application is as follows:

main_window = QtWidgets.QMainWindow()

Functions of the QMainWindow class

The QMainWindow class inherits QWidget and improves functionality by means of the following functions.

add

These are functions related to the addition of elements to the main window, such as the addition of a dock widget or toolbar:

main_window.addDockWidget(QtCore.Qt.DockWidgetArea, QtWidgets.QDockWidget, QtCore.Qt.Orientation): This will add the second parameter dock widget to the area of the first parameter in the direction specified by the third orientation parameter.

main_window.addDockWidget(QtCore.Qt.DockWidgetArea, QtWidgets.QDockWidget): This will add the dock widget of the second parameter to the area specified in the first parameter.

main_window.addToolBar("Title"): This method creates a QtWidgets.QToolBar object, setting the title, and inserts it into the top toolbar area.

main_window.addToolBar(QtWidgets.QToolBar): This will add a toolbar to the main window.

main_window.addToolBar(QtCore.Qt.ToolBarArea, QtWidgets.QToolBar): This will add the toolbar (the second parameter) to the specified area (the first parameter) in the main window.

main_window.addToolBarBreak(QtCore.Qt.ToolBarArea): This will add the toolbar break to the given area after objects that are present.

set

These are functions related to the setting of the parameters/properties in relation to the main window, including the setting of style, size, or functionality:

main_window.setAnimated(bool): If the parameter is True, this sets the main window as animated.

main_window.setCentralWidget(QtWidgets.QWidget): This sets the central widget specified in the parameter for this main window.

main_window.setCorner(QtCore.Qt.Corner, QtCore.Qt.DockWidgetArea): This sets the dock widget area (the second parameter) to occupy the corner (the first parameter).

main_window.setDockNestingEnabled(bool): If the parameter is True, this stipulates that docks can be nested.

main_window.setDockOptions(QtWidgets.QMainWindow.DockOptions): This sets the docking behavior of this main window.

main_window.setDocumentMode(bool): If the parameter is True, this sets the document mode to the tab bar of this main window (for tabbed dock widgets).

main_window.setIconSize(QtCore.QSize): This sets the icon size of the toolbar icons for this main window.

main_window.setMenuBar(QtWidgets.QMenuBar): This sets the menu bar specified in the parameter for this main window.

main_window.setMenuWidget(QtWidgets.QWidget): This sets the menu bar of the QtWidgets.QWidget type for this main window.

main_window.setStatusBar(QtWidgets.QStatusBar): This sets the status bar for this main window.

main_window.setTabPosition(QtCore.Qt.DockWidgetAreas, QtWidgets.QTabWidget.TabPosition): This sets the tab position for the dock widget areas (the first parameter) to the tab position (the second parameter).

main_window.setTabShape(QtWidgets.QTabWidget.TabShape): This sets the tab shape that will be used with tabbed dock widgets for this main window.

main_window.setToolButtonStyle(QtCore.Qt.ToolButtonStyle): This sets the style of toolbar buttons for this main window.

main_window.setUnifiedTitleAndToolBarOnMac(bool): This sets the unified title and toolbar look on macOS.

is

These are functions that return a Boolean value (bool) related to the main window states, for example, to check the current state:

main_window.isAnimated(): This returns True if the dock widgets and toolbars are animated.

main_window.isDockNestingEnabled(): This returns True if the docks can be nested.

functional

These are functions that are related to the returns of the current values of the main window, changes in functionality, and so on:

main_window.centralWidget(): This returns the central widget of the QtWidgets.QWidget type for this main window.

main_window.corner(QtCore.Qt.Corner): This returns the dock widget area that occupies the area specified in the parameter corner of the QtCore.Qt.Corner type.

main_window.createPopupMenu(): This returns a popup menu of the QtWidgets.QMenu type for this main window.

main_window.dockOptions(): This returns the docking behavior of the QtWidgets.QMainWindow.DockOptions type for this main window.

main_window.dockWidgetArea(QtWidgets.QDockWidget): This returns the dock widget area of the QtCore.Qt.DockWidgetArea type for the dock widget of the main window.

main_window.documentMode(): This returns True if the tab bar for tabbed dock widgets is set to document mode.

main_window.iconSize(): This returns the size of toolbar icons of the QtCore.QSize type in this main window.

main_window.insertToolBar(QtWidgets.QToolBar, QtWidgets.QToolBar): This inserts the toolbar (the second parameter) before the toolbar (the first parameter).

main_window.insertToolBarBreak(QtWidgets.QToolBar): This inserts a toolbar break before the toolbar specified in the parameter.

main_window.menuBar(): This returns the menu bar of the QtWidgets.QMenuBar type for the main window.

main_window.menuWidget(): This returns the menu bar of the QtWidgets.QWidget type for the main window.

main_window.removeDockWidget(QtWidgets.QDockWidget): This removes the dock widget from the main window layout and hides it.

main_window.removeToolBar(QtWidgets.QToolBar): This removes the toolbar from the main window layout and hides it.

main_window.removeToolBarBreak(QtWidgets.QToolBar): This removes the toolbar break before the toolbar specified in the parameter.

main_window.resizeDocks([QtWidgets.QDockWidget], [int], QtCore.Qt.Orientation): This resizes the dock widgets specified in the first parameter with sizes in pixels from the second parameter and a third orientation parameter.

main_window.restoreDockWidget(QtWidgets.QDockWidget): This restores the state of the dock widget with the QtWidgets.QMainWindow.restoreState() function and returns True if the state was restored; otherwise, it returns False.

main_window.restoreState(QtCore.QByteArray,0): This restores the state of the toolbars and dock widgets for this main window with the version as the second parameter of the integer value type that will be compared with the number stored in state. If they do not match the function, they return False, otherwise True.

main_window.saveState(0): This saves the current state of the toolbars and dock widgets for this main window. The version number of the parameter as an integer value will be stored as part of the data.

main_window.splitDockWidget(QtWidgets.QDockWidget, QtWidgets.QDockWidget, QtCore.Qt.Orientation): This splits the space covered by the widget (the first parameter) into two parts, moves the first dock widget into the first part, and moves the second dock widget (the second parameter) into the second part, with orientation from the third parameter that specifies how the space will be divided.

main_window.statusBar(): This returns the status bar of QtWidgets.QStatusBar for this main window.

main_window.tabifiedDockWidgets(QtWidgets.QDockWidget): This returns the dock widgets that are tabified together with dock widgets in the parameter.

main_window.tabifyDockWidget(QtWidgets.QDockWidget, QtWidgets.QDockWidget): This moves the dock widget in the second parameter on top of the dock widget from the first parameter and creates a tabbed docked area in the main window.

main_window.tabPosition(QtCore.Qt.DockWidgetArea): This returns the tab position for the area specified in the parameter.

main_window.tabShape(): This returns the tab shape of the QtWidgets.QTabWidget.TabShape type that is used with tabbed dock widgets.

main_window.takeCentralWidget(): This removes the central widget of the main window.

main_window.toolBarArea(QtWidgets.QToolBar): This returns the toolbar area of the QtCore.Qt.ToolBarArea type for the specified parameter.

main_window.toolBarBreak(QtWidgets.QToolBar): This returns True if the toolbar has a break before the toolbar specified in the parameter.

main_window.toolButtonStyle(): This returns the tool button style of the QtCore.Qt.ToolButtonStyle type.

main_window.unifiedTitleAndToolBarOnMac(): This returns True if the main window uses the unified title and toolbar look on macOS.

events

These are functions that are related to the events, such as event handlers:

main_window.contextMenuEvent(QtGui.QContextMenuEvent): This is the QWidget event handler, which is designed to receive main window context menu events.

main_window.event(QtCore.QEvent): This is the QtCore.QObject function, which receives events to the object and should return True if the event is recognized and processed.

signals

The available signals of the QMainWindow class are as follows:

main_window.iconSizeChanged(QtCore.QSize): This signal is emitted when the window's icon size has changed, with the new icon size that was passed in the parameter.

main_window.tabifiedDockWidgetActivated(QtWidgets.QDockWidget): This signal is emitted when the tabified dock widget is activated by selecting the tab, with the tabified dock widget that was passed in the parameter.

main_window.toolButtonStyleChanged(QtCore.Qt.ToolButtonStyle): This signal is emitted when the style that was used for tool buttons in the window is changed, and the new tool button style is passed in the parameter.

Summary

This chapter involved a detailed consideration of the two main classes of both the PySide2 and PyQt5 libraries: QWidget and QMainWindow.
