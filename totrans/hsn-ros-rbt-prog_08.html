Section 3: Autonomous Navigation Using SLAM

This section is purely about ROS and robotics engineering. You will acquire the skills to enforce ROS applications in order to execute navigational tasks. You will learn that path planning is essential for success and how this feature is built on top of the map of the covered area that's generated with Simultaneous Localization and Mapping (SLAM) techniques.

This section comprises the following chapters:

  • Chapter 6, Programming in ROS – Commands and Tools
  • Chapter 7, Robot Control and Simulation
  • Chapter 8, Virtual SLAM and Navigation Using Gazebo
  • Chapter 9, SLAM for Robot Navigation
